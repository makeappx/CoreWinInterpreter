# CoreWinInterpreter
Интерпретатор языка RedCode94

>Доступные типы адресации:
  * `$` - Прямая
  * `@` - Косвенная
  * `<` - Косвенная с предекрементом
  * `>` - Косвенная с постинкрементом

>Доступные команды:
  
`MOV, CMP, ADD, SUB, MUL, DIV, MOD, JMP, JMZ, JMN`
